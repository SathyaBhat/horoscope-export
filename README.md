# horoscope-export

## Requirements:

 - Running MySQL Server
 - python 2 & pip
 
 ## Install
 
    git clone git@github.com:SathyaBhat/horoscope-export.git
    pip install -r requirements.txt

 ## Run
 
    python export_horoscope.py -f <filename> -h <mysql host> -u <mysql username> -p <mysql password>
